Welcome to Anantam

Anantam is an innovative, next-generation IDE designed to empower developers and data enthusiasts. Where coding and data exploration possibilities are limitless.

🌟 Key Features

📊 Custom Chart System

Instantly generate interactive, real-time visualizations from your data. Touch-optimized charts with modern aesthetics.

Key Capabilities:

  • Automatic Chart Detection
  • Real-Time Interactivity
  • Custom Language Support (Infx)
  • Chart Export

Example (Python):

import pandas as pd

data = {'Year': [2020, 2021, 2022], 'Revenue': [100, 150, 200]}
df = pd.DataFrame(data)
# Anantam automatically detects and visualizes this DataFrame

📂 Global Variable System

Effortlessly share and access variables across your entire workspace.

Key Capabilities:

  • Seamless Integration with Files
  • IDE-Wide Availability
  • Manual Variable Creation (Infx)
  • Data Consistency

Example (Manual Variable Creation in Infx):

excel_df sales_data = excel_data("path/to/sales.xlsx")
csv_df revenue_stats = csv_data("path/to/revenue.csv")
json_df user_stats = json_data("path/to/users.json")
