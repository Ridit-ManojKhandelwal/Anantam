## Installation

To install AnantUI within Anantam, follow these steps:

```bash
  yarn add @mnovus/ui
```

## Usage

To use AnantUI components in Anantam, import them as follows:

```tsx
import { Button } from "@mnovus/ui";

const MyComponent = () => {
  return <Button>Click Me</Button>;
};

export default MyComponent;
```
